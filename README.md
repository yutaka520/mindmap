# シンプルマインドマップ

ブラウザ内に複数のマインドマップを保存できる、ローカルファーストの Next.js アプリです。TOPページからそのまま編集でき、保存先は Local Storage のままです。

## 起動

```bash
npm install
npm run dev
```

`http://localhost:3000` を開きます。

## Google Analytics

Google Analytics を有効にする場合だけ、`.env.local` などに測定 ID を設定してください。

```bash
cp .env.example .env.local
```

```env
NEXT_PUBLIC_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```

未設定の場合、GA スクリプトは読み込まれません。

## 主要ページ

- `/` : TOPページ。概要案内と埋め込みエディタを表示
- `/terms` : 利用規約
- `/privacy` : プライバシーポリシー

## マップ管理

左側のマップ一覧は保存単位です。各マップにはノード、エッジ、ノード位置、ズーム位置が個別に保存されます。

- `＋`：新しいプロジェクトを作成
- プロジェクト名をクリック：切替
- 名前を変更／マップを削除：選択中プロジェクトに対して実行

保存先はブラウザの Local Storage です。JSON 書き出し／読み込みは、選択中プロジェクトだけを対象にします。

## ノードとエッジの操作

- ノード四方の `＋` を**クリック**：その方向に新しいノードを追加し、自動で接続
- ノード四方の `＋` を**別ノードの＋までドラッグ＆ドロップ**：既存ノード同士をエッジ接続
- ノードをドラッグ：位置を移動
- エッジをクリック：選択。`Delete` / `Backspace` または右パネルのボタンで削除
- ノード本文をダブルクリック：名前を編集
- 空白をドラッグ：キャンバスを移動
- ホイール：拡大縮小

## 法務と計測

- 利用規約とプライバシーポリシーを App Router の固定ページとして追加しています
- プライバシーポリシーでは、Local Storage 保存と Google Analytics の利用条件を明記しています
- DB は使っておらず、マインドマップ本文はサーバーへ保存しません
